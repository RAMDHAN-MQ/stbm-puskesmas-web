# STBM Puskesmas Website
STBM Puskesmas Web (Admin Panel) adalah sistem berbasis web yang digunakan oleh admin/petugas Puskesmas untuk memantau, memverifikasi, dan mengelola data program Sanitasi Total Berbasis Masyarakat (STBM).

Website ini terhubung dengan aplikasi mobile dan berfungsi sebagai pusat kontrol dan monitoring data STBM.
